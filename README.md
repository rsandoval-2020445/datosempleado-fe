# Frontend - Sistema de Gestión de Colaboradores

Aplicación web desarrollada con React para la gestión de colaboradores.
Consume la API REST del backend y permite realizar operaciones CRUD mediante una interfaz gráfica.

Stack Tecnológico
- React
- Vite
- Material UI (MUI)
- React Hooks (useState, useEffect)

Requisitos
- Node.js (versión 20.x o superior recomendada)

Instalación y Ejecución

cd carpeta_de_destino
npm install
npm run dev

La aplicación estará disponible en la URL local proporcionada por Vite, normalmente:
http://localhost:5173

Funcionalidades
- Listado de colaboradores
- Creación, edición y eliminación de registros
- Consumo de API REST
- Diseño responsivo con Material UI
- Alertas y feedback visual al usuario
- Paginación del lado del cliente
- Manejo de errores y ruta 404 personalizada
